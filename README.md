# React Task Manager

This repository contains a task management frontend application built with React, TypeScript, and Vite. It allows users to create tasks, view task details, and delete tasks in a simple and intuitive interface.

## Technologies

<ul>
<li>Vite</li>

<li>React</li>

<li>TypeScript</li>

<li>ESLint</li>
</ul>

## Prerequisites

<ul>
<li>Node.js 14+</li>

<li>npm or yarn</li>
</ul>

## Instructions

1. Clone the repository

```
git clone https://github.com/analuizamelodev/vite-project.git
```

2. Install dependencies

```
npm install
```

3. Start the development environment

```
npm run dev
```

The application will run locally using Vite’s development server.

## Useful Scripts

`npm run dev` - Starts the development server
`npm run build` - Generates the production build (/dist)
`npm run preview` - Previews the production build locally
`npm run lint`(if available) - Runs ESLint to analyze the code
